## Especificações Técnicas

### Stack
- HTML5 semântico
- CSS3 (Flexbox, Grid, animations)
- JavaScript vanilla (ES6+)
- Sem frameworks ou dependências externas

### Design System
- **Cores:**
  - Fundo: #0a0a0f (preto profundo)
  - Secundário: #1a1a2e (cinza escuro)
  - Neon azul: #00d4ff
  - Neon roxo: #7c3aed
  - Alerta: #ef4444 (vermelho)
  - Sucesso: #22c55e (verde)
  - Texto: #ffffff, #94a3b8
- **Tipografia:**
  - Headlines: Inter ou similar (bold, 700-900)
  - Corpo: Inter ou similar (regular, 400-500)
- **Espaçamento:** 8px base (8, 16, 24, 32, 48, 64, 96)
- **Border radius:** 8px para cards, 12px para CTAs

### Animações
- **Hero:** Partículas flutuantes (canvas ou CSS)
- **Cards:** Glow effect no hover
- **Timer:** Pulsação sutil
- **Scroll:** Fade-in das seções (Intersection Observer)
- **CTA:** Hover com scale + glow

### Performance
- Lazy loading para imagens
- CSS animations (GPU accelerated)
- Minimal DOM manipulation
- Responsive images

### Mobile
- Breakpoints: 480px, 768px, 1024px, 1280px
- Cards empilhados em mobile
- CTA fixo no rodapé em mobile
- Timer responsivo

### Acessibilidade
- Contraste WCAG AA
- Alt texts em imagens
- Navegação por teclado
- Focus states visíveis
